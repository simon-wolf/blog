[YASnippet](https://github.com/joaotavora/yasnippet) is a template, or text expansion, system for Emacs. Whilst I don't tend to use a huge number of templates I do, for example, have one to create the metadata blocks my blog posts use:

```
---
date: 2026-04-11 12:00
title: 
categories: 
tags: 
---
```

As well as saving me from having to remember the structure of the metadata block the snippet does a couple of extra bits:

1. It automatically inserts the current date in a particular format. It could set the time too but I prefer to use a dummy, fixed time.

2. It provides me with pick-lists for values that go into the categories and tags.
